Ring Timer info

A simple, adjustable, pomodoro-style timer. Enjoy!

Each time the clock runs out, a check will appear to indicate a complete session, helping you keep track of how many pomodoro sessions you have completed in a day.

This app uses the latest CSS which might not work in some browsers yet. Sorry!
